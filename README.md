# Job4Students (.NET)

Job4Students is a desktop-based application developed using C# and .NET framework. The system is designed to connect students with job opportunities, making it easier to browse and manage part-time or entry-level job listings in a structured and user-friendly way.

## Features

- User registration and login system
- Student profile management
- Job listing and browsing functionality
- Apply for jobs directly from the system
- Admin panel for managing job posts and users
- Role-based access control (Student / Admin)
- SQL Server database integration
- Full CRUD operations for jobs and users

## Technologies Used

- C#
- .NET (Windows Forms / ASP.NET depending on implementation)
- SQL Server
- ADO.NET
- Visual Studio

## Project Purpose

The purpose of this project is to build a practical job management platform for students while improving skills in database handling, CRUD operations, authentication systems, and .NET application development.

## Future Improvements

- Email notification system for job applications
- Advanced job search and filtering
- Payment system for premium job posts
- Mobile application support
